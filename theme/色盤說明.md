# 全域 Theme 色盤說明（P8-2）

`theme/main_theme.tres` 是全專案的預設 Theme，透過 `project.godot` 的
`gui/theme/custom` 掛成 `ThemeDB.project_theme`——**所有 Control 自動繼承**
（不受 Node2D / CanvasLayer 祖先影響）。四場景 root 是 Node2D（無 theme 欄位），
故採「專案預設主題」而非逐 root 掛載，效果等同且更全面。

## 色盤

| 用途 | 色值 (RGB) | 位置 |
|---|---|---|
| 背景底色（深） | `0.09, 0.10, 0.13` | 各場景 Background ColorRect＋`default_clear_color` |
| 面板底 | `0.14, 0.16, 0.20` (α .98) | Panel styleBox |
| 面板/按鈕外框 | `0.30, 0.34, 0.42` | styleBox border |
| 主文字 | `0.92, 0.93, 0.95` | Label/Button font_color |
| 次要/說明文字（灰） | `0.70, 0.75, 0.82` | 逐節點 override（副標、說明、版本） |
| 停用文字 | `0.50, 0.53, 0.58` | Button font_disabled_color |
| 按鈕 normal 底 | `0.18, 0.20, 0.25` | btn_normal |
| 按鈕 hover 底/框 | `0.25, 0.29, 0.36` / `0.45, 0.55, 0.70` | btn_hover |
| 按鈕 pressed 底/框 | `0.13, 0.15, 0.19` / `0.40, 0.50, 0.65` | btn_pressed |
| 焦點框 | `0.50, 0.65, 0.85` (α .7) | btn_focus（draw_center=false）|
| 文字黑描邊 | `0, 0, 0` (α .9)、outline_size 4 | Label 預設，覆蓋各種底色皆可讀 |

**強調色（accent）** 目前不進 theme：各場景以逐節點 `font_color` override 表達語意
（P1 先手偏紅 `0.95,0.5,0.5`、P2 後手偏藍 `0.55,0.7,1`、警示橘 `1,0.7,0.5`、
提示藍 `0.75,0.85,0.95`）。若之後要統一，再抽成具名色進 theme。

## 派別色（不在此 theme 內）

10 色派別色是**資料驅動**，一律走 `Balance.color_rgb(card_id)`（來源 `data/balance`，
見 05）。棋子/卡片/圖鑑用它上色，**不要**在 theme 或程式硬編派別色。

## 字型插槽（之後換字型只改這裡）

目前用引擎內建字型（未設 `default_font`）。要換自訂字型：
把字型檔（.ttf/.otf → FontFile）放入專案，在 `main_theme.tres` 的 `[resource]` 段加：
```
default_font = ExtResource("<字型檔>")
```
並在檔頭 `[ext_resource ...]` 宣告該字型。全專案文字即一次換掉，無需動任何場景。

## 字級基準

- `default_font_size = 16`（未特別指定者）
- Button 預設 15、Label 預設 15；標題類（44/34/26/22/20…）以逐節點 `font_size` override。
