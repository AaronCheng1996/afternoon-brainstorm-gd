## 檢查項目

### 主選單
- [ ] 顯示標題、副標、平衡資料版本（左下角）。
- [ ] 五個項目：本機對戰、戰役模式（灰）、爬塔模式（灰）、設定、離開。
- [ ] 戰役/爬塔為灰色不可按（Phase 3–5 才開放）。

### 設定頁（持久化）
- [ ] 點「設定」開啟面板；有「戰鬥提示」「戰鬥動畫」兩個開關與「返回」。
- [ ] 切換開關文字即時更新（開/關）。
- [ ] **重啟保留**：關掉動畫（或提示）→ 關閉遊戲 → 重開 `main_menu.tscn` → 設定頁仍是關（存於 `user://settings.json`）。
- [ ] 由設定關掉動畫後進入對戰，對戰的動畫開關預設為「關（瞬時）」。

### 整條流程（不中斷）
- [ ] 主選單 →「本機對戰」→ 進入 **選秀 BP**（draft）。
- [ ] 完成 BP →「開始對戰」→ 進入 **對戰**，雙方牌組為 BP 所選。
- [ ] 對戰分出勝負 → 勝負面板出現三鈕：**再來一局／終局統計／回主選單**。
- [ ] 點「終局統計」→ 進入 **終局畫面**：顯示勝者＋最終分數、每回合分數折線（含 0 線與 ±門檻線）、
      主要統計長條（擊殺／造成傷害／得分 前幾名）。
- [ ] 終局畫面「回主選單」→ 回到 **主選單**（流程閉環）；「重新選秀」→ 回到 BP。
- [ ] 對戰勝負面板「回主選單」也能直接回到主選單。
