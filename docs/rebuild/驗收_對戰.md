## 如何執行
- 編輯器開啟 `scenes_v2/battle/battle.tscn`，按 **F6**。
- 預設開一局：雙方各一套 12 張牌組（含白/藍/綠/褐鏽/魅紫外的示範色，供顯示四種色資源列）。
- 本機雙人 hot-seat：同一畫面輪流操作，HUD 上方顯示「當前：先手 P1 / 後手 P2」。

## 操作對照
- **模式切換**：畫面右側按鈕或鍵盤 `A`=攻擊、`M`=移動、`H`=治療、`C`=方塊。
- **出牌**：點手牌卡 → 單位卡進入「放置中」→ 點棋盤空格放置；魔法卡（HEAL/MOVE/CUBES）點卡即生效（獲得次數）。
- **升級切換（Cyan）**：先點該 Cyan 手牌 → 按「升級切換(+)」。
- **攻擊/治療**：切到對應模式 → 點我方棋子。
- **移動（兩段式）**：切移動模式 → 點我方非暈眩棋子（啟用、扣移動點）→ 再點該子（選取）→ 點相鄰空格。
- **結束回合**：右下按鈕或 `Space`/`Enter`。
- **提示開關** `T`、**動畫開關** `I`、**重開** `R`。

## 檢查項目（R §4 每個行動至少操作一次）

- [ ] **出牌（單位）**：選單位卡 → 放到空格，棋子出現、外框顏色符合先/後手，入場為暈眩（有暈眩圖示）。
- [ ] **出牌（魔法 HEAL/MOVE/CUBES）**：點卡後對應「次數」HUD +1（HEAL+1 / MOVE+1 / CUBES+2）。
- [ ] **攻擊**：攻擊模式點我方棋子，命中範圍內敵人；攻擊次數 −1；有攻擊動畫＋飄字＋被擊閃爍＋血量下降。
- [ ] **攻擊範圍預覽**：攻擊模式滑鼠移到我方棋子上，會以紅色高亮其命中格（ADC 大十字＝同行同列、TANK/LF 小十字、ASS 斜角、HF 九宮、AP/APT/SP 依最近/最遠）。
- [ ] **移動（兩段式）**：解暈後的棋子可啟用移動（黃/藍高亮）、選取、移動到相鄰空格；不相鄰或被佔則失敗且移動點不退。
- [ ] **治療**：治療模式點我方受傷棋子回 6 HP；溢出轉護盾；治療次數 −1。
- [ ] **放方塊**：方塊模式點空格生成中立方塊（灰色，不計分）；方塊次數 −1。
- [ ] **升級切換**：Cyan 卡切換後名字出現「＋」；放置升級版需足夠金幣（金幣不足則放置失敗、牌留手上）。
- [ ] **結束回合**：換手，當前玩家指示與顏色更新；攻擊/移動/治療/方塊次數重置。

## 規則體感

- [ ] **暈眩（numbness）規則**：入場當回合為暈眩、該回合結算得 0 分、下回合開始解暈；攻擊指令對暈眩敵人仍可命中（暈眩是「不得分/不能行動」而非「不可被打」）。
- [ ] **分數方向**：P1 棋子得分使總分往負走、P2 往正走；HUD 顯示領先方與門檻 ±10。
- [ ] **資源列**：運氣/藍球/圖騰/金幣僅在雙方牌組含該色時顯示，且數值隨對局變動。
- [ ] **提示浮窗**：開啟提示時，滑鼠移到棋子上顯示該卡名稱＋能力提示；關閉後不顯示。
- [ ] **動畫開關**：關閉後行動瞬時完成（無逐格動畫），結果與開啟時一致。
