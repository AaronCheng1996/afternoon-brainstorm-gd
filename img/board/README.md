# 棋盤底圖插槽（P14-5）

對戰棋盤有兩種視角（對戰畫面按 **V** 或「視角」鈕切換），各吃一張底圖：

| 檔名 | 視角 | 對齊方式 |
|---|---|---|
| `skin_ortho.png` | 俯視（正交，方格） | 圖的**左上角**對齊 `battle.tscn` 的 `BoardAnchorOrtho` |
| `skin_iso.png` | 45 度（等距，菱形） | 圖的**上邊中點**對齊 `BoardAnchorIso`（菱形頂角） |

- 有圖 → 該視角的底圖顯示在格線之下、棋子之上遮不到；切視角會自動換成另一張。
- 沒圖 → 兩張都隱藏（**本目錄沒有 `skin_*.png` 時外觀與現況完全相同：只有格線**）。
- 底圖跟著 `BoardAnchor*` 走：在編輯器拖動 anchor，格線、棋子、高亮、底圖整組一起移動（P14-2）。
- 尺寸請照現行棋盤：4×4 格、格距 118px（俯視）／半寬 60、半高 48（45 度）；
  這些格距在 `battle.tscn` 根節點的「棋盤幾何」@export 可調。

本目錄既有的 `board_slot.png` / `hand_slot.png` / `selected_slot.png` 是舊 pygame 版的素材，
目前 Godot 版沒有使用。
